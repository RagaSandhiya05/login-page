# 🌐 Glassmorphism Login UI

A modern login page built using HTML & CSS with a clean glassmorphism design and aesthetic background.

##  Features

* Glass blur effect
* Gradient UI
* Username & Password fields
* Centered layout

##  Tech Stack

* HTML
* CSS
* Boxicons

##  Run

Open `index.html` in your browser.
